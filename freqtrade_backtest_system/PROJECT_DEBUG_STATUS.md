## ⚠️ 已知问题

### 1. 高优先级问题

#### 1.1 Freqtrade命令未找到
**问题:** 系统无法找到freqtrade命令
**影响:** 无法执行实际的回测
**解决方案:**
```bash
# 安装freqtrade
pip install freqtrade
# 或者配置freqtrade路径
```

#### 1.2 Session State序列化问题
**问题:** ExecutionScheduler对象无法序列化
**影响:** 页面刷新时状态丢失
**解决方案:** 重构调度器为可序列化对象

#### 1.3 Jupyter集成导入问题
**问题:** JupyterLabIntegration导入失败
**影响:** Jupyter功能受限
**状态:** 已添加fallback机制

### 2. 中优先级问题

#### 2.1 策略重复名称
**问题:** 不同路径下的同名策略导致key冲突
**状态:** 已修复 - 使用文件路径hash创建唯一key

#### 2.2 依赖包缺失
**问题:** 部分可选依赖包未安装
**影响:** 某些高级功能不可用
**解决方案:** 
```bash
pip install nbformat nbconvert streamlit-option-menu papermill jupyterlab
```
